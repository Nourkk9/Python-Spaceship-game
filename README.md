# Python Spaceship Game

A simple 2D spaceship game built with Python and Pygame. Includes player movement, shooting mechanics, enemies, sound effects, and basic game logic, with persistent saving of player profiles, scores, and level progress.

## Requirements
- Python 3.10+ (recommended)
- Pygame

## Setup
```bash
python -m venv .venv
# Windows PowerShell
.\.venv\Scripts\Activate.ps1
python -m pip install --upgrade pip
python -m pip install pygame
